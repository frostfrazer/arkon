Features

Everything you need to ship

Built for indie developers and small teams who want full control.

🔍

Auto-detection

Arkon detects your project type automatically — Next.js, Rust, Python, Go, Unity, Docker, and more.

🚀

Deploy anywhere

SSH servers, AWS S3, Cloudflare R2, Backblaze B2, GitHub Pages, IPFS, BitTorrent, Docker registries.

🔗

Instant preview links

Share a live preview URL with anyone in seconds — no server required, powered by P2P WebRTC.

🔐

Encrypted secrets vault

Store API keys and deploy credentials locally, encrypted, with BIP39 mnemonic recovery.

⏪

Rollback in one command

Every deploy saves a snapshot. Roll back to any previous version instantly.

🛡️

Audit log

Tamper-evident HMAC chain of every deploy — know exactly what shipped, when, and by whom.

🖥️

Desktop GUI

Native desktop app built with Tauri — not Electron. Fast, lightweight, cross-platform.

🧩

Community adapters

Install adapters from Git repos to add support for any project type without touching core.

🔒

Auto TLS

Automatic Let's Encrypt certificates via ACME when deploying to SSH targets. Zero config.

How it works

Ship in 3 steps

From zero to deployed in under a minute.

1

Install

One curl command installs arkon and configures the relay.

2

Init

Run arkon init in your project. Arkon detects everything and writes arkon.toml.

3

Ship

Run arkon ship. Arkon builds, deploys, and gives you a live URL.

terminal
$ arkon init
✓ detected: Next.js 14
✓ wrote arkon.toml

$ arkon ship --target s3
building artifact...
uploading 47 files...
✓ deployed in 12s
→ https://myblog.s3-website.amazonaws.com

$ arkon preview
✓ preview link ready
→ https://arkon-relay.onrender.com/p/abc123
